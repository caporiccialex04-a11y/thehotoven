# The Hot Oven

Marketing website for **The Hot Oven**, a from-scratch bakery in Verschoyle (southwestern Ontario), near Ingersoll.

Content, hours, contact details, and product photography are drawn from their [Facebook page](https://www.facebook.com/profile.php?id=100064419262638) and local coverage.

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
